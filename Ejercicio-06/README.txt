El funcionamiento de la calculadora Energetica consiste en realizar un calculo del consumo y de los costes utilizando una calculadora RPN. Habrá una serie de botones con distintos objetos con consumo del hogar donde se ingresará
el tiempo y los vatios. Se iran añadiendo objetos a la pila hasta que se selecione consumo donde calculará los kwh consumidos al mes y el coste. Ademas se podrá añadir un nº de personas en el hogar para calculos como el calentador
el cual el gasto se multiplicará por el nº de personas, (por defecto el programa tiene 2) y el precio de Kwh (por defecto está el del dia 22/11/2022).

Las teclas asociadas a los botones son las siguientes:
 - ['0'-'9'] 	-> Digitos
 - '.' 	 	-> Punto
 - '+' 	 	-> Suma
 - '-' 	 	-> Resta
 - '*' 	 	-> Multiplicacion
 - '/' 	 	-> Division
 - 'r' 	 	-> Raiz
 - 'l'   	-> log
 - 'l' + ALT 	-> ln
 - 'u'   	-> potencia cuadrado
 - 'y'   	-> xpotenciay
 - 's' + ALT   	-> seno
 - 'h'   	-> shift
 - 'c' + ALT   	-> coseno
 - ''   	-> tangente
 - 't' + ALT 	-> television
 - 'n' + ALT 	-> horno
 - 'd' + ALT 	-> radiador
 - 'e' + ALT 	-> calentador
 - 'p' + ALT 	-> precio
 - 'm' 		-> consumo
 - 'p'		-> nPersonas
 - 'i' 		-> iluminacion
 - 'v' 		-> lavavajillas
 - 'o'		-> ordenador
 - 'c'   	-> Ce
 - 's' 	 	-> Cambio de signo
 - 'ESC' 	-> On
 - 'Enter' 	-> Enter
