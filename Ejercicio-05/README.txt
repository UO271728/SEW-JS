El funcionamiento de la calculadora Milan consiste en introducir una serie de numero seguido de un operador y otra serie de numeros. Al igual que en una calculadora milan, ha medida que se realicen las operaciones
se irán apilando el resultado en un operador y el siguiente se irá introduciendo junto con el operador deseado. A parte habrá operaciones especiales como raiz o porcentaje que se calcularan sobre el ultimo numero que aparece en pantalla
o sobre el operando almacenado. Se podrá almacenar en memoria un valor el cual se podrá sumar valores en él, restar o mostrar el valor. 

Las teclas asociadas a los botones son las siguientes:
 - ['0'-'9'] -> Digitos
 - '.' 	 -> Punto
 - '+' 	 -> Suma
 - '-' 	 -> Resta
 - '*' 	 -> Multiplicacion
 - '/' 	 -> Division
 - '%' 	 -> Porcentaje
 - 'r' 	 -> Raiz
 - '=' 	 -> Igual
 - 'o' 	 -> On
 - 'c'   -> Ce
 - 's' 	 -> Cambio de signo
 - 'm' 	 -> M+
 - 'n' 	 -> M-
 - 'a'   -> MAC
 - 'ESC' -> On
 - 'Enter' -> Enter
